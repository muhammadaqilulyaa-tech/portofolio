Website Warkop Modern

Website Warkop Modern dengan tampilan minimalis, elegan, dan responsif yang dirancang untuk memberikan pengalaman terbaik bagi pelanggan. Website ini memiliki fitur menu interaktif, filter kategori makanan dan minuman, sistem keranjang belanja, pilihan varian seperti Hot & Iced, animasi modern, serta desain UI/UX yang clean dan nyaman dilihat di berbagai perangkat. UI Design

Aqil Payment — Fintech Mobile UI

Aqil Payment merupakan project simulasi aplikasi dompet digital modern berbasis Flask dan SQLite dengan tampilan mobile premium. Project ini dirancang menggunakan konsep UI fintech profesional yang menampilkan dashboard saldo, transaksi, top up balance, profile user, hingga navigasi interaktif dalam satu sistem aplikasi digital wallet. Desain interface menggunakan kombinasi modern gradient, glassmorphism, floating navigation, serta animasi halus untuk memberikan pengalaman visual yang elegan dan futuristik layaknya aplikasi fintech sungguhan.
